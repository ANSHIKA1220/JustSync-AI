# API

Interactive documentation is available at `/docs` when the backend is running.

Major groups:

- `/auth/login`, `/auth/me`
- `/customers`, `/customers/{id}/timeline`
- `/conversations`, `/messages`, `/tickets`
- `/analytics/summary`
- `/knowledge`, `/knowledge/search`, `/knowledge/reindex`
- `/ai/analyze`, `/ai/suggestions/{id}/approve`, `/ai/suggestions/{id}/reject`
- `/routing/rules`, `/routing/decide`
- `/audit`
- `/demo/reset`, `/demo/run-scenario`
- `/health`
