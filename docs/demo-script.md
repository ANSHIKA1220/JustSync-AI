# Demo Script

1. Start with `docker compose up --build`.
2. Log in as the support agent.
3. Open the dashboard and point out calculated CX metrics.
4. Open Unified Inbox and filter for high priority or negative sentiment.
5. Open Agent Workspace and select the damaged delivery case.
6. Explain omnichannel continuity: delayed delivery started in web chat, damaged delivery continued by email.
7. Review AI summary, intent, urgency, churn risk, retrieved sources, and routing recommendation.
8. Edit and approve the AI-generated response.
9. Open Audit & AI Transparency to show the human-in-the-loop record.
10. Run the guided demo scenario to reset and replay the main flow.
